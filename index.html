About This System

Project Context

This system was developed as a high-impact Capstone Project by specialists of the Machine Learning Path from the highly selective Asah by Dicoding and Accenture program. It aims to revolutionize interview assessments by providing an AI-powered solution that ensures fairness, accuracy, and transparency in evaluating candidate performance.              

Core Technology

Our system leverages cutting-edge Machine Learning models, including Whisper ONNX for robust Speech-to-Text, Computer Vision for non-verbal behavioral analysis, and custom rubric scoring algorithms for scalable and objective evaluation.

Key Features

  • 96% STT Accuracy powered by Whisper ONNX
  • Real-time Behavioral Analysis (Computer Vision)
  • Objective Integrity and Violation Detection
  • Detailed and Automated Rubric Scoring

Team

This AI-Powered Interview Assessment System is the result of a dedicated Capstone Project developed by Team A25-CS349 (Rajendriya Wigar Punagi, Arief Sidik Wijayanto, Rakha Syifa Athaya Zen). Driven by the Machine Learning Path's core commitment under the Dicoding and Accenture program, the team's mission is to revolutionize the hiring process by implementing a transparent, accurate, and fair AI system that analyzes candidate performance and integrity.
